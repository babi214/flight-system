# ✈️ Flight Management System (Django REST Framework)

A backend system for managing flights, users, bookings, and payments built using Django REST Framework with Clean Architecture principles.

---

## 🚀 Features

- 👤 User Registration & Authentication (JWT ready)
- ✈️ Flight Management (CRUD)
- 🎫 Booking System
- 💳 Payment Module (structure ready)
- 🔐 Role-based access (Admin/User)
- ⚡ Clean Architecture (services-based structure)
- 📦 Scalable app structure (users, flights, bookings, payments)

---

## 🏗️ Tech Stack

- Python 3.x
- Django
- Django REST Framework
- PostgreSQL / SQLite
- SimpleJWT (authentication)
- Redis (future caching support)
